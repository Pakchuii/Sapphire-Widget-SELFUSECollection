## ✦ Introduction

这是一个专为 **Sapphire** 桌面环境打造的自用组件包。

在这里，我们不满足于死板的背景图。通过 WebGL 与自定义物理引擎，我们试图在桌面的方寸之间，重现立绘应有的灵动感。

> [!NOTE]
> 🟡 **Notice**: 本仓库主要为自用存档，部分组件可能需要特定的 Sapphire 环境支持。
> 🔵 **Status**: 持续收录中，物理效果不断迭代。

---

## ✦ Featured: Embed_JiggleImage

**「 触手可及的柔软 —— 交互式多层物理组件 」**

这是库中最核心的部件，它不仅仅是一个展示窗，而是一个具备物理生命力的实体。

*   ✨ **Multi-Layer Physics**: 突破单一网格限制，支持为不同部位（头发、饰品、甚至...）配置独立的物理层。
*   🖌️ **Live Weight Painting**: 无需外部建模，直接在桌面上通过画笔涂抹权重，定义你的实时物理响应区。
*   🌬️ **Dual Simulation**: 同时支持鼠标互动的「摇动」与模拟生命的「呼吸」，让静止的图片活起来。
*   💾 **Project Management**: 完善的角色工程系统，一键切换你的每一位老婆，配置云端同步。
