## ✦ Introduction

这是一个专为Sapphir桌面美化软件打造的自用组件包。

> [!NOTE]
> 🟡 **Notice**: 本仓库主要为自用存档，部分组件可能需要特定的环境支持。
> 🔵 **Status**: 持续收录中，物理效果不断迭代。


## ✦ Components List

| 🔵 Status | Component | Description |
| :--- | :--- | :--- |
| 🟢 Active | **Embed_JiggleImage** |多图层实时物理涂抹与交互 |
| 🟢 Active | **Embed_MP4Player** | 对官方视频播放器的修复 |
| 🟢 Active | **Embed_GeminiChat** | 桌面端大模型组件（目前仅支持DS） |
| 🟢 Active | **Embed_WeatherCard** | 实时天气展示卡片 |
| 🟡 Maintenance | **Embed_StickyNote** | 便签，支持多色主题与即时保存 |
| 🔵 Ongoing | **Embed_BiliTrending** | B站热搜榜单实时同步 |
